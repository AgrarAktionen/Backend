REST - Endpunkte

Endpunkt Methode Beschreibung Body
/api/person GET Listet alle Personen auf -
/api/person/{id} GET Gibt die Person mit der gegebenen id zurück -
/api/person PUT Fügt die mitgesendete Person zur Personenliste hinzu Einzufügende Person als application/json
/api/person/{id} DELETE Löscht die Person mit der gegebenen id -
/api/person/download GET download aller Personen als text im toString() format -
/api/school GET Gibt alle Schulen zurück -
/api/school/{id} GET Gibt die Schule mit der gegebenen id zurück -
/api/html/persons GET Gibt Eine Tabelle aller Personen aus. -
/api/bill GET Gibt Eine Tabelle der Rechnungen aus. -
